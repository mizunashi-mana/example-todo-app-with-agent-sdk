# アプリのデモ動画を撮影

## 目的・ゴール

`playwright-cli` を使って TODO アプリのデモ動画を撮影し、リポジトリに保存する。
サンプルプロジェクトとしての完成度向上（Phase 5）の一環。

## 実装方針

1. `playwright-cli` のブラウザ操作 + `video-start` / `video-stop` でデモ動画を録画
2. TODO の作成・編集・削除、チャットによる自然言語操作など主要機能をデモ
3. 録画した動画をリポジトリに配置

## 完了条件

- [x] デモ動画が撮影されリポジトリに保存されている
- [x] 主要機能（TODO CRUD、チャット操作）がデモされている

## 作業ログ

### テイク1（手動 UI 中心）
- 手動 UI の CRUD 操作を中心にデモ撮影 → チャットメインに方針変更

### テイク2（チャット中心）- 採用
- `playwright-cli` で `http://localhost:5173` を開き、video-start で録画開始
- チャットでの自然言語操作をメインにデモ:
  1. addTodo x3: 「買い物リストを作って。牛乳、パン、卵の3つ」→ 3つの TODO が作成された
  2. listTodos: 「今のTODO一覧を見せて」→ 一覧が表示された
  3. toggleTodoStatus: 「牛乳は買ったから完了にして」→ 完了済みに変更
  4. deleteTodo: 「卵はやっぱりいらないから削除して」→ 削除された
- 動画を `docs/assets/demo.webm`（2.1MB）として保存
