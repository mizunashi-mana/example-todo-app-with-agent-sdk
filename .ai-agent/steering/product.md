# プロダクトビジョン

## ミッション

Ollama と Agent SDK を使い、自然言語で操作できる TODO タスク管理アプリの実践的なサンプルを提供する。
ローカル LLM を活用したエージェントアプリの設計パターンと実装方法を示す。

## ターゲットユーザー

- Ollama + Agent SDK を使った開発を始めたい開発者
- AI エージェントをアプリケーションに組み込む方法を学びたいエンジニア

## 課題

- Agent SDK の公式サンプルだけでは、実際のアプリケーションでの使い方がわかりにくい
- Ollama とエージェントフレームワークの連携パターンが確立されていない
- 断片的なコード例では、本番品質のプロジェクト構成を学べない

## ユースケース

- 自然言語で TODO タスクを作成・編集・削除・完了する
- エージェントがユーザーの意図を理解し、適切な操作を実行する
- Agent SDK と Ollama の連携パターンをコードベースから学ぶ

## 主要機能

- TODO タスクの CRUD 操作（作成・読取・更新・削除）
- 自然言語による TODO 操作インターフェース
- Ollama ローカルモデルによるエージェント処理
- 開発者向けの設計パターン参考実装

## 差別化ポイント

- クラウド API 不要、Ollama だけで完結するエージェントアプリ
- TODO アプリという普遍的なドメインで Agent SDK の理解に集中できる
- devenv による再現可能な開発環境
