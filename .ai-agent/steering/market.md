# 市場分析

## 市場概要

ローカル LLM を活用した AI エージェント駆動アプリケーションのサンプルプロジェクト市場に位置する。
Ollama + Agent SDK による実践的な学習リソースとしての役割を持つ。

## ターゲットセグメント

- ローカル LLM（Ollama）を使ったエージェントアプリ開発を検討している開発者
- AI エージェントを組み込んだアプリケーションの構築パターンを学びたい開発者
- クラウド API に依存しない AI アプリケーション開発に関心のある開発者

## 競合分析

| プロジェクト | 特徴 |
|---|---|
| 各 Agent SDK 公式サンプル | フレームワーク固有、最小限のコード例が中心 |
| LangChain テンプレート | LangChain エコシステム特化、複雑になりがち |
| 各種ブログ記事・チュートリアル | 断片的、メンテナンスされない |

## 差別化ポイント

- Ollama（ローカル LLM）に特化し、クラウド API 不要で動作する
- TODO アプリという馴染みのあるドメインでエージェントの動作を理解しやすい
- 本番品質のコード構成（lint、テスト、CI）を備えた学習リソース

## 市場動向

- ローカル LLM の性能向上と普及（Llama, Mistral, Gemma 等）
- AI エージェントフレームワークの急速な進化（2025〜2026年）
- プライバシー・コスト面からローカル LLM への関心の高まり
